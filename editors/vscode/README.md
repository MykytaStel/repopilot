# RepoPilot for VS Code

This preview extension is distributed as platform-specific VSIX files attached
to RepoPilot GitHub Releases. It is not currently published to the VS Code
Marketplace.

The extension runs the bundled local RepoPilot binary. It provides change
review and workspace scan commands, Problems diagnostics, Markdown reports, and
registers the RepoPilot MCP server for VS Code agent mode.

It does not scan on save, upload source, send telemetry, or run in untrusted
workspaces.
