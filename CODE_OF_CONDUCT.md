# Code of Conduct

RepoPilot is maintained as a professional open source project.

## Expected Behavior

- Be respectful and constructive.
- Keep discussions focused on the project and the technical problem.
- Give actionable feedback when reviewing code or documentation.
- Assume public project spaces are shared by people with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Repeatedly derailing technical discussions.
- Publishing private information without permission.

## Enforcement

Maintainers may remove comments, close issues or pull requests, or block participants who violate this code of conduct.
